# TLC Photography — Logo Generation Prompts for Gemini

Use these prompts in Gemini (or any image generation model). Each prompt includes full brand context so the model understands the aesthetic target. After generating raster concepts you like, convert to vector using one of these local/free methods:

- Adobe Illustrator → Image Trace (you have Creative Suite)
- OmniSVG local model (free, image-to-SVG mode, runs via ComfyUI or standalone)
- StarVector local model (free, image-to-SVG, excels at logotypes and icons)

Export final logos in: SVG, PNG (transparent, multiple sizes), PDF, EPS, ICO, WEBP


---

## Prompt 1: Botanical Wordmark (Primary Logo — Lavender & Sage)

```
Design a refined, elegant wordmark logo for "TLC Photography," a boutique photography studio in a small southern US town. The logo should include a small, delicate botanical element — a single honeysuckle-inspired sprig with 2-3 small tubular flower buds and a few tiny paired leaves on a thin curving stem. The botanical element should be subtle and integrated near or between the letters, not overpowering the typography.

Typography direction: Use a refined, high-contrast serif typeface similar to Cormorant Garamond — thin strokes with elegant contrast, classic and airy. The word "TLC" should be prominent with "Photography" smaller beneath or beside it.

Color palette: Warm charcoal (#3A3338) for the text, with the botanical element in muted sage green (#C2CBB2) and soft lavender (#9B8EC4) for the flower buds. Background should be transparent or warm cream (#FAF7F4).

Style: Clean, minimal, sophisticated. NOT playful, NOT rustic, NOT heavy. Think fine stationery, not Pinterest craft. The botanical element should look hand-drawn but precise — like a botanical illustration, not a cartoon flower. The overall feeling should be serene, southern, elevated. This is a brand that photographs love stories and family legacies.

Output on a clean white or cream background. Show the full logo at a scale where fine details are visible.
```

## Prompt 2: Botanical Wordmark (Primary Logo — Golden Hour)

```
Design a refined, elegant wordmark logo for "TLC Photography," a boutique photography studio in the American South. Include a small, delicate botanical element — a single honeysuckle-like sprig with 2-3 tiny tubular flower buds and a few small paired leaves on a thin curving stem. The botanical element should be subtle and positioned near or between the letters.

Typography direction: Use a warm, slightly quirky optical-size serif typeface similar to Fraunces — rounder letterforms with soft wedge serifs, warm and handcrafted feeling but still professional. "TLC" prominent with "Photography" smaller beneath or beside it.

Color palette: Deep walnut brown (#2F2623) for the text, botanical element in olive sage (#A3AB8E) with flower buds in dusty rose (#C08B76) and golden honey (#C9A45C) accents. Background should be transparent or warm ivory (#FBF6F0).

Style: Warm, organic, artisanal but elevated. NOT trendy, NOT corporate, NOT heavy-handed. Think letterpress printed on textured paper. The botanical element should feel like a pressed flower specimen — delicate, natural, precise. This brand photographs golden-hour portraits and family heirlooms.

Output on a clean ivory or white background at a scale where fine details are visible.
```

## Prompt 3: TLC Monogram with Botanical (Icon/Favicon)

```
Design a compact monogram logo using the letters "T", "L", and "C" for a boutique photography brand called "TLC Photography." The three letters should be composed into a single cohesive mark that works at small sizes (as small as 32x32 pixels).

Add a tiny botanical accent — a single small sprig with 1-2 honeysuckle-like flower buds — growing from or near the top of the letterforms. The botanical element must be extremely small and simple at this scale.

Typography: The letters should use a refined serif typeface with elegant contrast (similar to Cormorant Garamond). The letters can overlap, interlock, or stack — whatever creates the most balanced and readable composition.

Show two versions:
1. Dark version: Letters in warm charcoal (#3A3338) with sage green (#C2CBB2) botanical detail and lavender (#9B8EC4) flower buds, on transparent/cream background
2. Light version: Letters in warm cream (#FAF7F4) on a circular lavender (#9B8EC4) background with cream botanical details

Style: Refined, minimal, works at small scale. NOT busy, NOT ornate. The monogram should be instantly readable as "TLC" and the botanical element should be a whisper, not a shout.
```

## Prompt 4: Pure Wordmark — Stacked (No Botanical)

```
Design a clean, typography-only wordmark for "TLC Photography." No icons, no symbols, no graphic elements — pure type.

Layout: "TLC" large on top, "Photography" smaller beneath, centered. A thin horizontal line or minimal divider between them is acceptable.

Typography: Use a refined high-contrast serif typeface with thin strokes and classical proportions (similar to Cormorant Garamond or Didot). "TLC" should have generous letter-spacing (tracked out). "Photography" should be in a clean, light-weight sans-serif (similar to Raleway Light) in small caps or uppercase with wide letter-spacing.

Color: Warm charcoal (#3A3338) on transparent or warm cream (#FAF7F4) background. If using a divider line, make it lavender (#9B8EC4).

Style: Extremely clean, editorial, confident. This should look like the masthead of a high-end photography magazine. No decoration whatsoever. The beauty comes from the typeface selection, spacing, and proportions alone.

Show one version and make it flawless.
```

## Prompt 5: Botanical Wordmark — Horizontal (Full Width)

```
Design a horizontal wordmark for "TLC Photography" with a small botanical accent. The entire logo should read left to right on a single line, suitable for website headers and business cards.

"TLC" in a refined serif (Cormorant Garamond style), then a small honeysuckle sprig (2-3 tiny tubular buds on a thin stem with 2 small leaves), then "Photography" in a lighter weight sans-serif or italic serif.

The botanical sprig acts as a natural separator between the two words. It should be the same height as the lowercase letters — small, not dominating.

Colors: Text in warm charcoal (#3A3338), stem and leaves in sage (#C2CBB2), flower buds in soft lavender (#9B8EC4). Cream or transparent background.

Style: Balanced, horizontal, elegant. Should work at widths from 200px to 600px without losing legibility. The botanical element should feel like a natural breath between the words, not an afterthought pasted in.
```

## Prompt 6: Exploration — Vine Wrapping the L

```
Design a wordmark for "TLC Photography" where a delicate honeysuckle vine subtly wraps around or grows from the vertical stroke of the letter "L" in "TLC."

The vine should be extremely delicate — a single thin stem with 2-3 tiny paired leaves and 1-2 small tubular flower buds. It should look like the vine is naturally growing along the letter, not overlaid on top of it. The vine integrates with the typography rather than sitting beside it.

"TLC" in a refined, high-contrast serif typeface (Cormorant Garamond style). "Photography" smaller beneath in a light sans-serif.

Colors: Charcoal text (#3A3338), sage vine (#C2CBB2), lavender buds (#9B8EC4), cream background (#FAF7F4).

Style: This should feel like finding a pressed flower inside an old book — quiet, natural, discovered rather than designed. The vine should be so subtle that you might miss it at first glance, then notice it and appreciate the detail.

Make it look like a real designer spent weeks refining the integration between the letterform and the botanical element. This is fine craftsmanship, not clip art.
```

## Vectorization Pipeline (All Local/Free)

After selecting the best Gemini raster outputs:

1. Clean up in Photoshop if needed (remove artifacts, sharpen edges)
2. Convert raster to vector using one of these local options:

**Option A: Adobe Illustrator Image Trace** (you already have this)
- Open raster PNG in Illustrator
- Object → Image Trace → "High Fidelity Photo" preset
- Expand, then manually simplify paths
- Best control, most tedious

**Option B: OmniSVG (local model, free)**
- GitHub: https://github.com/OmniSVG/OmniSVG
- Model weights: https://huggingface.co/OmniSVG/OmniSVG1.1_4B (lighter) or OmniSVG1.1_8B (better quality)
- Supports: text-to-SVG, image-to-SVG, and character-reference SVG
- Runs standalone (Python + CUDA) or via ComfyUI node (see below)
- Requirements: NVIDIA GPU (8GB+ VRAM for 4B, 16GB+ for 8B), Python 3.10, CUDA 12.1
- Install: conda env, pip install, Cairo library
- ComfyUI node: https://github.com/smthemex/ComfyUI_OmniSVG
- Built on Qwen-VL, handles up to 30,000 token sequences for complex SVGs

**Option C: StarVector (local model, free)**
- GitHub: https://github.com/joanrod/star-vector
- Model weights: https://huggingface.co/starvector/starvector-8b-im2svg
- Specifically trained on logotypes, icons, technical diagrams (not photos)
- Two sizes: StarVector-1B (lighter) and StarVector-8B (best quality)
- Apache 2.0 license, accepted at CVPR 2025
- Includes Gradio web UI for easy use
- Best choice specifically for logo vectorization since it was trained on that task


## Required Export Formats

Once you have the final vector logo, export from Illustrator:

| Format | Use Case | Specs |
|--------|----------|-------|
| SVG    | Web, responsive display | Clean paths, no embedded rasters |
| PNG    | Social media, digital | Transparent bg, export at 500px, 1000px, 2000px, 4000px widths |
| PDF    | Print, documents | Vector-based, CMYK color space |
| EPS    | Print shops, legacy | Vector, outlined text |
| ICO    | Favicon | 16x16, 32x32, 48x48 multi-size |
| WEBP   | Web performance | Transparent bg, multiple sizes matching PNG |

Export each logo variant (primary botanical wordmark, stacked wordmark, horizontal wordmark, monogram) in all formats above, for both palettes (Lavender & Sage + Golden Hour). That's roughly 48 files total. Store in a `/logos/` directory organized by variant and palette.

See `ComfyUI-Full-Build.md` in this folder for the complete ComfyUI toolkit setup, including SVG/vector nodes for logo vectorization.
